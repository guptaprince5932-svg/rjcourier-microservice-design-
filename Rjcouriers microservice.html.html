Microservices Overview

Independent, scalable services working together

API Gateway

Single entry point for all client requests, handling routing, authentication, and rate limiting.

GET /api/v1/*

User Service

Manages user authentication, profiles, and authorization across the platform.

POST /api/v1/users/login

Order Service

Handles package creation, management, and order processing workflows.

POST /api/v1/orders

Tracking Service

Provides real-time package location updates and delivery status information.

GET /api/v1/tracking/{id}

Notification Service

Sends email, SMS, and push notifications for order updates and alerts.

POST /api/v1/notifications

Payment Service

Processes payments and handles transaction management securely.

POST /api/v1/payments
